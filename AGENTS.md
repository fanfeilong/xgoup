## Project type

This repository is an **XGo** project.

- **Source of truth**: `*.xgo`
- **Generated output**: `xgo_autogen.go` (generated by `xgo build`)

`xgo_autogen.go` files are **not committed**. If you edit `*.xgo`, you must regenerate before building/testing.

## Development workflow

- **Generate**:

```bash
xgo build ./...
```

- **Build / test**:

```bash
go test ./...
```

## Notes

- CI should run `xgo build ./...` before `go test ./...`.
- When reviewing changes, focus on `*.xgo` diffs; `xgo_autogen.go` should not appear in PRs.
