# Windows helpers

- `xgoup.ps1` / `xgoup.cmd`: legacy bootstrap wrappers (optional; prefer the unified `xgoup` binary).
- **Install** is handled by the single `xgoup` executable: run `xgoup self install` (see project `README.md` / `README.windows.md`).

The former separate `xgoup-init` tool has been removed; its behavior lives under `xgoup self install`.
